# ☁️ Cloud Storage App (Google Drive Clone)

A full-stack **Cloud Storage Application** inspired by Google Drive, designed to allow users to upload, store, manage, and access files securely from anywhere.

This project demonstrates real-world implementation of file handling, backend integration, and full-stack development.

---

## 🔗 Live Links

🌐 **Live Application (Render):**  
👉 https://cloud-storage.onrender.com  

📂 **GitHub Repository:**  
👉 https://github.com/yogijangir345/cloud-storage  

---

## 🚀 Key Features

- 🔐 User Authentication (Login / Signup system)
- 📁 Upload files to cloud storage
- 📂 View and manage uploaded files
- 🗑️ Delete files securely
- ⬇️ Download files anytime
- 🌐 Access from any device/location
- ⚡ Fast backend response
- 🔒 Secure file handling & API structure

## ⚙️ Installation & Setup Guide

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/yogijangir345/cloud-storage.git
```

### 2️⃣ Navigate to Project Folder
```bash
cd cloud-storage
```

### 3️⃣ Install Dependencies
```bash
npm install
```

### 4️⃣ Start Backend Server
```bash
npm start
```

### 5️⃣ Run Frontend (if separate)
```bash
npm run dev
```

## 📬 API Endpoints

| Method | Endpoint        | Description              |
|--------|----------------|--------------------------|
| GET    | /files         | Fetch all uploaded files |
| POST   | /upload        | Upload a new file        |
| DELETE | /file/:id      | Delete a file            |
| GET    | /download/:id  | Download file            |
